# 로또 예측 분석기 - 크롬 확장프로그램

로또 645 당첨 번호 통계를 분석하여 예상 번호를 제안하는 크롬 확장프로그램입니다.

## 주요 기능

### 1. 데이터 수집
- 동행복권 사이트에서 최신 로또 당첨 번호 자동 수집
- 최근 10회, 20회, 50회, 100회차 선택 가능
- 로컬 스토리지에 데이터 캐싱 (24시간 유효)

### 2. 다양한 분석 기법

#### 빈도 분석
- 각 번호(1-45)의 출현 빈도 계산
- 가장 자주 나온 번호 Top 10 표시
- 시각적 막대 차트로 표현

#### 핫/콜드 넘버 분석
- **핫 넘버**: 최근 회차에서 자주 나온 번호
- **콜드 넘버**: 오랫동안 나오지 않은 번호
- 마지막 출현 회차 추적

#### 패턴 분석
- **홀짝 비율**: 홀수/짝수 출현 비율 분석
- **구간별 분포**: 1-10, 11-20, 21-30, 31-40, 41-45 구간별 분석
- **연속 번호**: 연속된 번호 출현 패턴 분석
- **번호 합계**: 당첨 번호 합계의 범위와 평균 계산

#### 종합 통계
- 전체 분석 회차 수
- 최다/최소 출현 번호
- 평균 홀짝 비율
- 연속 번호 출현율

### 3. AI 기반 예상 번호 생성
여러 분석 기법을 종합하여 점수 기반으로 예상 번호 6개를 생성합니다:
- 빈도 분석 가중치
- 핫 넘버 우선순위
- 콜드 넘버 역발상
- 구간별 균형 조정
- 홀짝 비율 최적화

## 설치 방법

### 1. 프로젝트 다운로드
```bash
git clone https://github.com/muzbox1973/lotto_chrome.git
cd lotto_chrome
```

### 2. 아이콘 파일 준비
아이콘이 필요합니다. `icons/README.md` 파일을 참고하여 아이콘 파일을 생성하세요.

간단한 방법:
1. `icons/icon.svg` 파일을 https://cloudconvert.com/svg-to-png 에서 변환
2. 16px, 32px, 48px, 128px 크기로 각각 저장
3. `icons/` 폴더에 `icon16.png`, `icon32.png`, `icon48.png`, `icon128.png`로 저장

### 3. 크롬에 확장프로그램 로드

1. 크롬 브라우저를 열고 주소창에 `chrome://extensions/` 입력
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택
5. 확장프로그램이 설치되면 브라우저 툴바에 아이콘이 표시됩니다

## 사용 방법

1. 크롬 툴바에서 확장프로그램 아이콘 클릭
2. 분석할 회차 수 선택 (10회, 20회, 50회, 100회)
3. "데이터 수집 및 분석" 버튼 클릭
4. 잠시 기다리면 분석 결과와 추천 번호가 표시됩니다
5. 탭을 전환하며 다양한 분석 결과 확인

### 분석 탭 설명

- **빈도 분석**: 전체 회차에서 가장 많이 나온 번호 확인
- **핫/콜드 분석**: 최근 트렌드 파악
- **패턴 분석**: 홀짝 비율, 구간별 분포 등 패턴 확인
- **통계 정보**: 종합 통계 데이터 확인

## 프로젝트 구조

```
lotto_chrome/
├── manifest.json           # 확장프로그램 설정 파일
├── popup.html             # 팝업 UI
├── styles.css             # 스타일시트
├── popup.js               # 팝업 UI 제어 스크립트
├── lotto-data.js          # 데이터 수집 모듈
├── lotto-analyzer.js      # 분석 알고리즘 모듈
├── background.js          # 백그라운드 서비스 워커
├── icons/                 # 아이콘 파일들
│   ├── icon.svg          # SVG 아이콘 (변환용)
│   ├── icon16.png        # 16x16 아이콘
│   ├── icon32.png        # 32x32 아이콘
│   ├── icon48.png        # 48x48 아이콘
│   └── icon128.png       # 128x128 아이콘
└── README.md             # 프로젝트 설명서
```

## 기술 스택

- **Manifest V3**: 최신 크롬 확장프로그램 표준
- **Vanilla JavaScript**: 프레임워크 없이 순수 JavaScript 사용
- **Chrome Storage API**: 로컬 데이터 저장
- **Fetch API**: 로또 데이터 수집
- **CSS3**: 현대적인 UI 디자인

## 주의사항

⚠️ **면책 조항**
- 이 프로그램은 통계 기반 분석 도구입니다
- 생성된 번호는 참고용이며, 당첨을 보장하지 않습니다
- 로또는 확률 게임이므로 모든 번호 조합이 동일한 당첨 확률을 가집니다
- 책임 있는 구매를 권장합니다
