# 아이콘 파일 안내

이 디렉토리에는 크롬 확장프로그램 아이콘 파일들이 필요합니다.

## 필요한 파일

- icon16.png (16x16 픽셀)
- icon32.png (32x32 픽셀)
- icon48.png (48x48 픽셀)
- icon128.png (128x128 픽셀)

## 아이콘 생성 방법

### 옵션 1: SVG를 PNG로 변환

1. 이 디렉토리에 있는 `icon.svg` 파일을 사용하세요
2. 온라인 도구를 사용하여 PNG로 변환:
   - https://cloudconvert.com/svg-to-png
   - https://svgtopng.com/
3. 각각 16px, 32px, 48px, 128px 크기로 변환하여 저장

### 옵션 2: 온라인 아이콘 생성기 사용

1. https://www.favicon-generator.org/ 방문
2. 로또 공 이미지 또는 숫자 이미지 업로드
3. 다양한 크기의 아이콘 다운로드
4. 이 디렉토리에 저장

### 옵션 3: 직접 디자인

Photoshop, GIMP, Figma 등을 사용하여 직접 디자인할 수 있습니다.

## 임시 해결방법

아이콘이 없어도 확장프로그램은 작동하지만, 크롬에서 경고가 표시될 수 있습니다.
임시로 간단한 단색 아이콘을 사용하려면:

1. 온라인 PNG 생성기 (https://png-pixel.com/) 사용
2. 보라색(#667eea) 배경에 흰색 숫자 "7" 이미지 생성
3. 필요한 크기로 저장
